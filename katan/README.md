My City Gov
===

---

Clone the repository:

```shell
git clone https://github.com/Ropoperperithras/katan
```

---

Run the example:

```shell
cd katan
./mvnw spring-boot:run  # MacOS / Linux
./mvnw.cmd spring-boot:run  # Windows
```

---

Open in browser: [localhost:8080/test](http://localhost:8080/test)
